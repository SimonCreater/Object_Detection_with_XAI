# raw/ — 자료 투입구 (여기에 당신의 자료를 넣으세요)

이 폴더는 위키 페이지의 **원문 소스 보관함**입니다. 논문 PDF, 메모(.md/.txt), 정리 노트 등
위키로 통합하고 싶은 자료를 여기에 넣은 뒤, 에이전트에게 통합을 요청하면 됩니다.

## 사용법 (3단계)

1. 자료를 이 폴더에 복사한다. 예: `raw/my_paper.pdf`, `raw/my_note.md`
2. 저장소 루트에서 Claude Code 를 열고 요청한다:
   ```
   /new-wiki-page raw/my_note.md
   ```
   또는 자연어로 — "raw에 넣은 my_note.md 위키 페이지로 통합해줘"
3. 에이전트가 원문 정독 → 중복 검색 → 페이지 생성 → 스키마 검증까지 마치고 slug 를 알려준다.
   웹 뷰어(`py tools/app.py` → http://127.0.0.1:5000)에서 새 페이지를 확인한다.

## 규칙

- 이 폴더의 원문은 **읽기 전용**으로 취급된다 — 에이전트가 수정/삭제하지 않는다 (RULES.md 6절).
- **PDF 등 원문 파일은 git 에 커밋되지 않는다**(`.gitignore`의 `raw/*.pdf`) — 저작권·용량 보호.
  위키 페이지가 `source.ref: raw/<파일명>` 으로 가리키므로, 로컬에 원문이 있으면 추적 가능.
- 예시로 `example_note.md` 가 들어 있다 — 처음 사용해 볼 때 이 파일로 1번 절차를 연습해 보세요.

## 이 위키의 기존 페이지 소스

기존 11개 페이지 중 7개는 실제 논문 7편(PDF)을 정독해 작성되었다.
어떤 페이지가 어떤 원문에서 왔는지는 `wiki/_meta/source_index.md` 에 매핑되어 있다.
